Теория (вопросы)

1. Что такое цикл? 
Цикл - это конструкция, позволяющая выполнять один участок кода несколько раз.

2. Что такое шаг или итерация цикла?
Шаг цикла/итерация - это одно выполнение тела цикла.

3. Какие типы циклов вы знаете?
Типы циклов:
- while - цикл с предусловием;
- do..while - цикл с постусловием;
- for - цикл со счетчиком;
- foreach.. - цикл “для каждого…”, разновидность цикла for

4. Какова синтаксическая структура каждого типа циклов (кроме цикла foreach..)?
while:

while (<условие>) {
    <тело цикла>
}

do..while:

do {
    <тело цикла>
} while (<условие>);

for:
for (<начальное действие>; <условие>; <действие после итерации>) {
    <тело цикла>
}

5. Что располагается в параметрах “начальное действие” и “действие после итерации” в цикле for?
- в параметрах “начальное действие” - обычно объявляется счётчик цикла, 
но может быть произведено любое другое действие. Данная часть выполняется только один раз, перед началом цикла.
- в параметрах “действие после итерации” - действие, которое будет выполнено после каждой итерации (шага), 
если она не была прервана. В этом месте обычно увеличивается счетчик цикла, 
который был объявлен в параметре “начальное действие”. Но может быть совершено и любое другое действие.

6. Что такое область видимости переменной?
Область видимости переменной - это свойство переменной быть доступной из тех или иных частей кода.

7. Каковы особенности области видимости переменной, объявленной в в параметре “начальное действие” цикла for?
Особенность заключается в том, что переменная будет доступна только в пределах цикла.

8. Для чего нужны операторы break и continue?
Оператор break нужен для прерывания цикла на определенном шаге до его завершения, а опреатор continue - для пропуска шага и перехода к следующему.

9. Какие способы задания бесконечного цикла вы знаете?
Если мы хотим, чтобы наш цикл не прекращался никогда, то этого можно добиться с помощью бесконечного цикла. 
Его можно создать, использую цикл while, do..while и for.
Для while  и do..while - используя true в условии, в for - не задавая параметров.
